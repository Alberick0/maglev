## Installing

To install, just clone the repo and do `make install`

```bash
git clone git@github.com:caiogondim/maglev.git
cd maglev
make install
```

Some dependencies:
- [Tmux](http://tmux.github.io/)
- [Powerline patched font](https://github.com/powerline/fonts)
